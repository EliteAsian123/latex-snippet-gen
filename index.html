          In order to use the exported .sty file in your LaTeX document,
          put the file in the same directory as your .tex file,
          and import it using \usepackage.


\documentclass{article}

\usepackage{./cmd}

\begin{document}
...


          Alternatively, you can copy and paste the contents of the .sty
          file at the top of your LaTeX document.
          


          Note that the .sty file always imports the xparse,
          amsmath, amssymb packages (along with the additional packages
          you selected).
